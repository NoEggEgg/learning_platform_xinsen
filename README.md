# 安全生产题库学习平台

一个基于 Web 的安全生产知识学习平台，帮助用户系统地学习和巩固安全生产相关知识。

## 功能特点

### 📚 学习计划
- 三阶段学习路径：基础学习 → 强化记忆 → 冲刺复习
- 智能进度追踪，实时显示学习进度
- 连续学习天数统计，激励用户坚持学习

### 📖 题库学习
- 支持多种题型：单选题、判断题
- 分类学习：法律类、法规类、国标类
- 即时反馈与详细解析
- 记忆技巧提示，帮助加深理解

### 📝 习题本
- **错题本**：自动记录答错的题目，方便复习
- **收藏夹**：收藏重点题目，便于反复练习
- 支持批量清空错题和收藏

### 🎯 模拟考试
- 60分钟限时考试
- 60道题目，满分100分，60分合格
- 实时倒计时与进度显示
- 考试结束后自动统计成绩并记录错题

### 💡 记忆技巧
- 提供记忆方法和学习技巧
- 帮助用户高效记忆知识点

## 技术架构

```
├── index.html          # 主页面入口
├── data/
│   └── questions.js     # 题库数据
└── js/
    ├── app.js           # 主应用逻辑
    ├── config.js        # 配置文件
    ├── state.js         # 状态管理
    ├── storage.js       # 本地存储
    ├── ui.js            # 用户界面控制
    ├── utils.js         # 工具函数
    ├── study.js         # 学习模块
    ├── exam.js          # 考试模块
    ├── achievement.js   # 成就系统
    ├── learning-path.js # 学习路径
    └── spaced-repetition.js # 间隔重复算法
```

## 核心功能模块

### 1. 学习计划模块
- 阶段解锁机制，循序渐进
- 实时进度可视化
- 今日学习目标追踪

### 2. 题库学习模块
- 题目筛选与过滤
- 答题反馈与解析
- 学习进度持久化

### 3. 模拟考试模块
- 计时考试系统
- 自动批改与评分
- 错题自动加入错题本

### 4. 状态管理
- 基于 LocalStorage 的数据持久化
- 学习进度、错题、收藏同步保存

### 5. 用户界面
- 响应式设计，支持移动端
- 现代化 UI 设计
- Toast 通知系统

## 快速开始

### 环境要求
- 现代浏览器（Chrome、Firefox、Safari、Edge）
- 无需服务器，纯前端运行

### 运行方式

1. **直接打开**
```bash
# 使用浏览器直接打开 index.html
open index.html
```

2. **本地服务器（推荐）**
```bash
# 使用 Python 启动本地服务器
python -m http.server 8000

# 或使用 Node.js
npx serve .

# 访问 http://localhost:8000
```

## 配置说明

主要配置项位于 `js/config.js`：

```javascript
{
  EXAM_DURATION: 3600,      // 考试时长（秒）
  EXAM_QUESTION_COUNT: 60,  // 考试题目数量
  EXAM_TOTAL_SCORE: 100,    // 总分
  EXAM_PASS_SCORE: 60,      // 及格分数
  TOAST_DURATION: 3000      // 提示显示时长
}
```

## 数据结构

### 题目数据
```javascript
{
  id: "q001",
  type: "single",        // single: 单选题, judge: 判断题
  category: "法律类",     // 法律类、法规类、国标类
  content: "题目内容",
  options: ["选项A", "选项B", "选项C", "选项D"],
  answer: 0,             // 正确答案索引
  memoryTip: "记忆技巧",
  analysis: "答案解析"
}
```

### 用户进度
```javascript
{
  streak: 7,             // 连续学习天数
  totalCompleted: 150,   // 已完成题目数
  studyTime: 120,        // 累计学习时间（分钟）
  wrong: ["q001", "q002"], // 错题ID列表
  favorite: ["q003"],    // 收藏ID列表
  examCompleted: 3       // 完成考试次数
}
```
