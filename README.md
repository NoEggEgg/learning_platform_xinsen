# 安全生产题库学习平台

一个基于 Web 的安全生产知识学习平台，帮助用户系统地学习和巩固安全生产相关知识。支持 500+ 题库，涵盖法律类、法规类、国标类三大类别。

## ✨ 功能特点

### 📚 智能学习计划
- **三阶段学习路径**：基础学习 → 强化记忆 → 冲刺复习
- **阶段解锁机制**：循序渐进，完成前一阶段自动解锁下一阶段
- **智能进度追踪**：实时显示各阶段学习进度百分比
- **连续学习天数统计**：激励用户坚持学习，记录学习里程碑
- **今日目标系统**：根据阶段动态调整每日学习目标

### 📖 题库学习（核心功能）
- **多题型支持**：单选题、判断题
- **分类筛选**：按类别（法律/法规/国标）和题型精准过滤
- **🎲 题目随机显示**：每次进入学习或下一题时，剩余题目随机打乱顺序
- **🎲 选项随机打乱**：每道题的答案选项位置随机排列，防止死记硬背
- **即时反馈与详细解析**：提交后立即显示正确与否及完整解析
- **记忆技巧提示**：提供助记方法，帮助加深理解
- **会话自动恢复**：未完成的学习进度自动保存，下次打开可继续
- **答题状态追踪**：已做题目标记状态，上一题/下一题导航智能管理

### 📝 习题本
- **错题本**：自动记录答错的题目，支持重做和清空
- **收藏夹**：收藏重点题目，便于反复练习
- **标签页切换**：错题和收藏独立管理
- **搜索功能**：快速定位特定题目
- **批量操作**：一键清空错题或收藏

### 🎯 模拟考试
- **60分钟限时考试**：真实模拟考试环境
- **60道综合题目**：从全题库随机抽取
- **100分制评分**：60分及以上合格
- **实时倒计时与进度显示**
- **自动批改与成绩统计**
- **错题自动加入习题本**

### 🎨 用户体验优化
- **响应式设计**：完美适配桌面、平板、手机
- **Toast通知系统**：中上位置显示，不遮挡答题区域
- **紧凑布局**：最大化答题区域空间
- **流畅动画**：页面切换、按钮交互均有动画反馈
- **深色模式友好**：使用 CSS 变量，易于主题定制

## 🔧 核心功能详解

### 1. 智能学习引擎
```
┌─────────────────────────────────────────┐
│           startStudy() 入口             │
├─────────────────────────────────────────┤
│  ① 检查是否有未完成的会话？              │
│     ├─ 有 → resumeStudySession() 恢复   │
│     └─ 无 ↓                            │
│  ② 初始化新会话                         │
│     ├─ shuffleArray() 随机题目          │
│     ├─ 生成 questionOrder 映射表       │
│     └─ 重置 answeredIds / questionStates │
│  ③ loadQuestion() 加载当前题            │
│     └─ shuffleOptions() 打乱选项        │
└─────────────────────────────────────────┘
```

### 2. 答题流程
```
选择答案 → submitAnswer()
    ↓
记录到 answeredIds + questionStates
    ↓
显示解析 + 更新进度 (XX%)
    ↓
自动跳转下一题（1.5秒后）
    ↓
nextQuestion()
    ├─ 在 visitedIndices 中前进？
    │   └─ 是 → 返回历史题目
    └─ 否 → 随机新题（跳过已答）
```

### 3. 会话持久化
```javascript
studySession: {
    active: Boolean,         // 会话是否活跃
    type: 'normal',         // 会话类型
    questionIds: [],         // 题目ID列表
    questionOrder: [],       // 随机顺序映射
    visitedIndices: [],      // 已访问的索引栈
    currentIndex: Number,    // 当前位置
    answeredIds: [],         // 已答题目ID
    correctCount: Number,    // 正确数
    wrongCount: Number,      // 错误数
    startedAt: Timestamp,    // 开始时间
    questionStates: {}       // 每题状态 {shuffledOptions, shuffledAnswer}
}
```

## 🚀 快速开始

### 环境要求
- 现代浏览器（Chrome 90+、Firefox 88+、Safari 14+、Edge 90+）
- **无需服务器**，纯前端运行
- **无需安装依赖**

### 运行方式

**方式一：直接打开**
```bash
# 双击 index.html 或在终端执行
start index.html        # Windows
open index.html         # macOS
xdg-open index.html     # Linux
```

**方式二：本地服务器（推荐）**
```bash
# Python 3
python -m http.server 8000

# Node.js (需要先安装 serve)
npx serve .

# 访问 http://localhost:8000
```

## ⚙️ 配置说明

配置文件：[js/config.js](js/config.js)

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `EXAM_DURATION` | 3600 | 考试时长（秒），默认60分钟 |
| `EXAM_QUESTION_COUNT` | 60 | 考试题目数量 |
| `EXAM_TOTAL_SCORE` | 100 | 总分 |
| `EXAM_PASS_SCORE` | 60 | 及格分数线 |
| `TOAST_DURATION` | 3000 | Toast通知显示时长（毫秒） |
| `PHASES` | - | 三阶段配置（名称、目标分类、题目数） |
